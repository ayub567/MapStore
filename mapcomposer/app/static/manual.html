How to Start a new run

  1. Click on the near the Row in the Algorithms table
    or
    Right click on a run and click on " Start from Here"
  2. Click on "Ok" button to start editing the execution input parameters
  3. Edit the form fields and click on "Execute"
  4. Wait for the process execution Ending

See results

A successful run execution row contains the link to the generated map. Click on the button to see the map of the results.

If the process failed, the button will show you details about the error occurred

Share your data

the Visibility column show what users can see the results of the process. The visibility can be

  • Private:Only the owner can see the run execution information
  • Shared:Only some selected groups can see the results
  • Public:Everyone can see the results.

If you are the owner of the process, you can share with other people the results of your run. Click on the To share the results with everyone or only with a group of users
